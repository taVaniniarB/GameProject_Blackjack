## Jack's BlackJack Game
개발 일자: 2023.05.23

간단한 콘솔 기반 블랙잭(BlackJack) 게임입니다. 플레이어와 딜러가 카드를 뽑아 21에 가까운 점수를 만드는 게임으로, 기본적인 블랙잭 규칙을 구현했습니다.
 프로젝트 개요
* 언어: Java
* 플랫폼: Console Application

## 게임 규칙
1. 게임 시작 시 플레이어와 딜러는 각각 2장의 카드를 받습니다.
2. 딜러의 카드 중 하나는 숨겨진 상태(XX) 로 표시됩니다.
3. 플레이어는 다음 중 선택할 수 있습니다:
   * Hit (h) : 카드 한 장 추가
   * Stand (s) : 카드 추가 없음
4. 플레이어 턴 종료 후 딜러 턴 진행
   * 점수가 17 이상이면 Stand
   * 17 미만이면 Hit
5. 결과 판정:
   * 21 초과 → Bust (패배)
   * 더 높은 점수 → 승리
   * 같은 점수 → 무승부

## 주요 기능
1. 카드 시스템
* 4가지 무늬 (♠, ♣, ♥, ◆)
* 13개의 숫자 및 그림 카드
* A는 기본 11, 필요 시 1로 변경
2. 점수 계산
* 숫자 카드: 숫자 그대로 점수
* J, Q, K: 10점
* A: 11점 또는 1점 (Bust 방지)
3. 랜덤 카드 분배
* 이미 뽑힌 카드는 다시 나오지 않도록 처리
* 2차원 배열을 이용한 카드 관리
4. 플레이어 / 딜러 행동
* 플레이어는 직접 Hit / Stand 선택
* 딜러는 자동으로 규칙에 따라 행동
5. 게임 종료 조건
* Bust 발생
* 21 도달
* 플레이어 Stand 후 결과 비교
6. 데이터 저장
* 게임 종료 시 `lastgame.txt`에 플레이어와 딜러의 잔액이 저장되며, 다음 실행 시 `2번`으로 이어서 플레이할 수 있습니다.

```
lastgame.txt 예시:
150
9850
```
